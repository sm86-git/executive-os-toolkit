# Weekly Exec Staff outputs checklist

## Decisions
- [ ] Decision statement written in plain language
- [ ] Decision owner and decision maker recorded
- [ ] Rationale captured (1-2 sentences)
- [ ] Follow-on actions assigned

## Actions
- [ ] Action has a single accountable owner
- [ ] Action has a due date
- [ ] Success is defined (what done looks like)

## Escalations
- [ ] The team is clear on what help is needed
- [ ] Options were presented (at least two)
- [ ] Next check-in date set

## Communications
- [ ] What to communicate and to whom
- [ ] Owner for the message
- [ ] Timing and channel
